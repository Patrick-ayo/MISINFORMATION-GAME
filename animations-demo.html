Animations Demo (explanations + code)

1 — Navbar Sparkle (CSS keyframes)

A small SVG 'sparkle' scales in using the @keyframes sparkle animation. This is triggered on hover and staggered via CSS variables.

SPARKLE

2 — Logo Letter Hover (transform + transition)

Each letter scales on hover using transform: scale() with a transition for smooth motion.

M I S I N F O

3 — Nav Glare (background-clip sheen)

A gradient is clipped to the text and moved on hover to create a sliding glare effect.

GLARE

4 — Chat Button Float (keyframes)

A simple bobbing effect based on @keyframes applied with animation: floatWave 3s ease-in-out infinite.

5 — Spinner (keyframes)

Continuous rotation using @keyframes spin.

6 — Hover button & placeholder (transitions)

Simple hover scaling and shadow for buttons and placeholders.

Placeholder card

7 — Hexagon Canvas (basic example)

A very small canvas demo that draws hexagons and responds to the pointer — uses requestAnimationFrame to continuously redraw.

8 — requestAnimationFrame demo (rotating square)

Shows how rAF drives a per-frame update loop similar to simple game loops.

This demo is local; open animations-demo.html in a browser. See comments in animations-demo.js for implementation notes.
